No inicio, a base de dados tem este tamanho;

A sistema de gestao de dados começa com os livros que existem
actualmente na biblioteca, que vao sendo povoados...
A base de dados da biblioteca vai depender do estado da bibliteca,
pois apenas vao ser inseridos nesta base de dados os livros que existirem
na biblioteca geral da universidade do minho.

Análise do seu crescimento: 
Sendo esta uma base de dados para um biblioteca, espera-se que esta
base de dados não se mantenha estática, mas sim, tenha cada vez mais
dados que vao sendo adicionados e alterados, ao longo do funcionamento 
da biblioteca. Por exemplo, quando chegam novos livros para a biblioteca, estes vão ser registados como novos, sendo adicionados ao povoamneto inicial da base de dados.
Quando um utilizador requisita um livro, o estado do livro é mudado e a sua disponibilidade passa a ser "requisitado"... 
Esperamos que esta base de dados cresca mas não de uma forma exponencial, mas sim, com pouco declive, mas sempre crescendo...
A não ser que hajam livros já "outdated" que vão ser arquivados.
